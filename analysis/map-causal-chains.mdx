`map_causal_chains`

This function maps how directional signals emerge at the highest frequency and propagate through a hierarchy of timeframes. By identifying the origin of a change, it constructs causal chains to quantify how micro-scale shifts evolve into structural change across expanding time scales.
---

#### Function Parameters

| Parameter  | Type     | Required | Description                                                                                                                                 |
| :--------- | :------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| data_input | `list[tuple]` |True| A list of tuples containing the source (URL or local path) and the associated timeframe label (e.g. ('path/to/file.csv', '1m').|
